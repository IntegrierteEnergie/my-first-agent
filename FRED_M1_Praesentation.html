Folie 03 · Konzept
Chatbot vs. Agent
Einfacher Chatbot
Frage rein → Antwort raus. Kein Gedächtnis. Keine Werkzeuge. Kein Plan. Jede Antwort ist eine Einbahnstraße.
KI-Agent
Plant Schritte. Nutzt Werkzeuge. Erinnert sich. Trifft Entscheidungen. Arbeitet iterativ auf ein Ziel hin.
R
Reasoning
Plant und bewertet Optionen
A
Action
Ruft Werkzeuge auf
M
Memory
Erinnert Kontext und History

Folie 08 · app.py
Streamlit in 5 Minuten
Warum Streamlit?
Reines Python. Kein HTML/CSS nötig. Chat, Graphen, Tabellen, Datei-Upload – alles dabei.
Für Studenten
Eine Datei. Eine Zeile zum Starten: streamlit run app.py
Session State
st.session_state hält den Agent über Interaktionen hinweg.
Nächste Schritte
Grafiken · Tabellen · Karten. Alles nachrüstbar ohne Umbau.
